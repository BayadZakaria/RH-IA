## 🚀 Projet Phare : EVOLIA (RH with IA Analysis)
**Evolia** est une application SaaS moderne conçue pour transformer le processus de recrutement grâce à l'intelligence artificielle, du dépôt de candidature à l'analyse prédictive.

### ✨ Fonctionnalités Clés
- 🧠 **Analyse de CV par IA** : Extraction automatique des compétences et scoring des candidats.
- 📂 **Gestion de Base de Données** : Centralisation des candidatures via Supabase (PostgreSQL).
- 🔐 **Authentification Sécurisée** : Connexion fluide via Google OAuth.
- 📊 **Dashboard Intuitif** : Vue d'ensemble des talents pour les décideurs RH.
- ⚡ **Performance** : Architecture optimisée avec Next.js pour une expérience utilisateur sans faille.
